# open-teaching-datasets

> Learning to work with cancer data — survival analysis, differential expression, variant annotation, staging logic, epidemiology, messy-data cleaning — is gatekept by a data problem. The realistic data  ·  **Risk tier:** low  ·  **Status:** planning

Learning to work with cancer data — survival analysis, differential expression, variant annotation, staging logic, epidemiology, messy-data cleaning — is gatekept by a data problem. The realistic datasets are **controlled-access** (dbGaP, EGA, individual-level biobanks) and take months of IRB/DAC approval a student or self-learner cannot obtain; the freely-downloadable ones are either toy tables with no cancer context, or large real archives whose license, provenance, and de-identification status are murky and whose size makes them unusable in a 90-minute lesson. The result: people who want to learn cancer data science practice on iris and titanic, then meet real oncology data cold.
